# 加盟店ごとの入力差し替えガイド（monet FC 融資パッケージ）

新規加盟店ごとに、以下の項目だけを差し替えれば提出書類一式が量産できる。
**申請者は必ず加盟店本人（フランチャイジー）。グランフュージョンはFC本部として記載。**

## 1. 名義・基本情報（build_xlsx.py の A2ヘッダー / build_docx.js の表紙）
- 申請者名義：氏名（個人事業主 or 法人名）
- 形態：個人事業主／法人
- 店舗名：monet ○○店
- 開業予定日／所在地
- FC本部：株式会社グランフュージョン（固定）

## 2. 設備資金（build_xlsx.py S1 items / build_docx.js 6.1表）
青セル（入力値）のみ差し替え：
- ① 内装・設備工事（見積根拠を注記）
- ② デザイン設計費
- ③ その他諸経費・備品代
- ④ 給湯機（ボイラー）
- ⑤ シャンプー設備
- ⑥ FC加盟金（繰延資産・5年償却）※FC共通額
- ⑦ 物件取得費（敷金/礼金/仲介/保証の内訳を注記）

## 3. 運転資金 / 借入
- ⑧ 運転資金
- 借入総額（= 開業資金総額 − 自己資金）
- 年利（既定2.5%）／返済年数

## 4. 前提条件（build_xlsx.py S2）
- 客単価・席数・回転・営業日数 → 月商
- 人件費・家賃・リース料（シャンプー台/マッサージチェア）・損害保険料
- ロイヤリティ率（既定5%・開業4ヶ月目から）

## 5. 代表者プロフィール（build_docx.js 2.2）
- 美容師歴／実績数値／受賞歴／グループ役職 等

## 6. 検証（必須）
- Excel：xlsxスキル recalc.py で数式エラー0
- Excel⇔Word：開業資金総額・借入額・毎月返済額・Y1経常利益が一致（pdftotext + grep）
- 結合PDF：merge_pdf.py（ghostscript /ebook 圧縮）
