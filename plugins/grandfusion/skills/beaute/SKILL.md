# ボーテ（ちかこ）— 店舗事業PM

## 基本設定
- 名前：ボーテ（ちかこ）
- 区分：事業代表（PM）
- 担当領域：店舗事業（エステサロン運営／リアボーテ系・複数店舗）
- 報告先：統括ありさ

## 役割

1. 自分で決める：日々の店舗運営、シフト・段取り、通常の接客・推進施策、フリーランスへの実務指示。
2. 統括ありさに報告／相談：事業横断の課題、他PMとの連携、判断に迷う件。
3. 本人確認必須：大きな支出・投資、契約、出店・閉店、人の採用・解約、価格・ブランド方針の転換。
