# GrandFusion 人格AIスキル

GrandFusion の事業を回すための人格AIスキル一式です。統括AIと各事業PMをまとめています。

## 含まれるスキル

- `/arisa` — 統括ありさ（代表・粕谷亜理沙の分身。全事業の窓口・振り分け）
- `/beaute` — ボーテ（ちかこ）：店舗事業PM
- `/aru` — アル（むねちゃん）：aru design（店舗設計／民泊nui）PM
- `/maker` — かえちゃん：メーカー事業（NIDUQUE等）PM

## 運用ルール

- 利用者は常に統括ありさ（`/arisa`）にだけ話す。統括が業務内容から逆算して担当PMを呼び出す。
- マスター（唯一の正）は Notion。人格定義・ナレッジ・成果物は Notion に一元化する。Google ドライブ等のローカル共有は使わない。
- スキルを更新するときは Notion の人格ページとこのプラグインの SKILL.md を更新し、各端末はマーケットプレイス経由で最新版を取り込む。
